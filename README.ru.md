# RTGame

RTGame — это многопользовательская игра реального времени, разработанная в качестве практического проекта для NGP Soft.

## Архитектура

Проект использует архитектуру клиент-сервер:

- **Сервер**: Приложение на C# .NET 10.0 с базой данных PostgreSQL
- **Клиент**: Приложение на C++ с SFML для графики и ImGui для пользовательского интерфейса

## Возможности

- Многопользовательская игра в реальном времени
- Система аккаунтов с аутентификацией
- Матчмейкинг на основе комнат
- Система миров/карт
- Сетевой протокол с поддержкой TCP и UDP
- Предсказание и интерполяция на стороне клиента
- Синхронизация состояния на основе снимков (snapshots)

## Технологии

**Сервер:**
- .NET 10.0
- Npgsql (драйвер PostgreSQL)
- Собственный сетевой протокол

**Клиент:**
- C++
- SFML (графика, окно, сеть)
- ImGui (пользовательский интерфейс)
- Менеджер пакетов vcpkg

## Структура проекта

```
RTGame/
├── Server/             # Серверное приложение на C#
│   ├── Accounts/       # Управление аккаунтами
│   ├── Database/       # Слой базы данных
│   ├── Maps/           # Игровые карты
│   ├── Network/        # Сетевой протокол
│   └── World/          # Логика игрового мира
└── Client/             # Клиентское приложение на C++
    ├── ImGui UI
    ├── Сеть (TCP/UDP)
    ├── Рендеринг
    └── Игровая логика
```

## Сборка

**Сервер:**
```bash
cd Server
dotnet build
```

**Клиент:**
```bash
cd Client
vcpkg install
# Сборка через Visual Studio или MSBuild
```

## Запуск

1. Настройте базу данных PostgreSQL
2. Настройте параметры подключения к серверу
3. Запустите сервер
4. Запустите клиент и подключитесь к серверу
